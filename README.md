# Smart Home Dashboard

A beautiful smart home dashboard for Home Assistant.

## Features

- 🏠 Room-by-room control
- 💡 Light control
- 🌡️ Thermostat
- 📹 Camera feeds
- 🔔 Doorbell integration
- 📊 Energy monitoring

## Demo

![Dashboard Preview](preview.png)

## Installation

1. Clone this repo
2. Edit `config.js` with your Home Assistant URL
3. Open `index.html` in browser

## Configuration

```javascript
const config = {
    haUrl: 'https://your-ha-url.ui.nabu.casa',
    token: 'YOUR_LONG_LIVED_TOKEN'
};
```
